# Vulkan Tutorial — Worked Implementation

A complete C++ implementation following the 
[Vulkan Tutorial](https://docs.vulkan.org/tutorial/) by Alexander Overvoorde,
built using the Vulkan RAII (`vk::raii`) C++ API rather than the C API
used in the original.

## What's covered
- Window surface, swapchain, image views
- Render passes and graphics pipeline
- Vertex/index buffers, staging buffers
- Descriptor sets, UBOs
